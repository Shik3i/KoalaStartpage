Privacy Policy

1. Hosting & Logfiles 1. Hosting & Logfiles

Diese Seite wird auf einem privaten Server gehostet. Zur Gewährleistung der Stabilität werden standardmäßige Server-Logs (IP-Adresse, Browser-Kennung, Zeitpunkt) erhoben, aber nicht mit Personen verknüpft und nach 7 Tagen automatisch gelöscht.

This site is hosted on a private server. To ensure stability, standard server logs (IP address, browser identifier, time) are collected, but not linked to individuals and are automatically deleted after 7 days.

2. Externe Dienste & API-Verbindungen 2. External Services & API Connections

Diese Webseite wird vollständig autark betrieben. Alle Schriftarten (Google Fonts Inter), Icons (Phosphor Icons) und Stylesheets werden lokal auf unserem Server gehostet. Es erfolgt beim Laden der Seite keine automatische Verbindung zu externen Content Delivery Networks (CDNs).

This website is operated completely independently. All fonts (Google Fonts Inter), icons (Phosphor Icons), and stylesheets are hosted locally on our own server. When loading the page, no automatic connection is made to external Content Delivery Networks (CDNs).

To dynamically display software versions in the dashboard, your browser asynchronously fetches data via the following official programming interface (API):

  • GitHub REST API — api.github.com

Dabei wird Ihre IP-Adresse an GitHub (GitHub Inc., USA) übermittelt. Es gelten die Datenschutzrichtlinien von GitHub.

When doing so, your IP address is transmitted to GitHub (GitHub Inc., USA). GitHub's privacy policy applies.

3. Lokale Speicherung 3. Local Storage

Die Seite speichert Ihre Sprachpräferenz (DE/EN) sowie einen temporären Cache der GitHub-Release-Daten im localStorage Ihres Browsers. Diese Daten verlassen Ihr Gerät nicht und können jederzeit über die Browser-Einstellungen gelöscht werden.

The site stores your language preference (DE/EN) and a temporary cache of GitHub release data in your browser's localStorage. This data never leaves your device and can be deleted at any time via your browser settings.

4. Cookies 4. Cookies

Diese Webseite verwendet keine Cookies.

This website does not use any cookies.

5. Kein Tracking 5. No Tracking

KoalaStuff verzichtet bewusst auf Analyse-Tools, Tracking-Cookies oder Werbenetzwerke.

KoalaStuff deliberately avoids analytics tools, tracking cookies, or advertising networks.

Sie haben das Recht auf Auskunft, Berichtigung oder Löschung Ihrer Daten. Da wir jedoch keine personenbezogenen Daten dauerhaft speichern, ist eine Zuordnung zu Ihrer Person in der Regel technisch nicht möglich.

You have the right to information, correction, or deletion of your data. However, since we do not store any personal data permanently, linking data to your person is technically impossible in most cases.
