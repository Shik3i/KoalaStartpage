Datenschutzerklärung

1. Verantwortlicher

Verantwortlich für die Datenverarbeitung auf dieser Webseite ist der Betreiber der Domain koalastuff.net. Kontaktdaten können dem Impressum entnommen werden.

2. Allgemeine Hinweise

Diese Webseite ist eine statische Startseite (Dashboard) ohne Benutzerkonten, Kontaktformulare oder Tracking-Dienste. Es werden keine personenbezogenen Daten erhoben, verarbeitet oder gespeichert, die über die technisch notwendige Auslieferung der Seite hinausgehen.

3. Hosting & Server-Logs

Die Seite wird über einen eigenen Webserver (Caddy) ausgeliefert. Beim Zugriff auf die Seite können folgende Daten in Server-Logs protokolliert werden:

  • IP-Adresse des anfragenden Geräts
  • Datum und Uhrzeit der Anfrage
  • Angeforderte Ressource (URL)
  • HTTP-Statuscode
  • User-Agent (Browser-Kennung)

Diese Daten werden ausschließlich zur Sicherstellung des technischen Betriebs und zur Fehleranalyse verwendet. Eine Zusammenführung mit anderen Datenquellen findet nicht statt.

4. Externe Dienste (CDN)

Diese Seite bindet folgende externe Ressourcen ein:

  • Tailwind CSS via cdn.tailwindcss.com
  • Phosphor Icons via cdn.jsdelivr.net
  • Inter Font via fonts.googleapis.com
  • GitHub REST API via api.github.com

Beim Laden dieser Ressourcen wird Ihre IP-Adresse an die jeweiligen CDN-Betreiber übermittelt. Es gelten die Datenschutzrichtlinien der jeweiligen Anbieter.

5. Lokale Speicherung

Die Seite speichert ausschließlich Ihre Sprachpräferenz (DE/EN) im localStorage Ihres Browsers. Diese Daten verlassen Ihr Gerät nicht und können jederzeit über die Browser-Einstellungen gelöscht werden.

6. Cookies

Diese Webseite verwendet keine eigenen Cookies.
